# FixIt Egypt — Project Overview

A one-page orientation for anyone picking up this project. For full detail, follow the links.

## What it is

**FixIt Egypt** is a two-sided **home-services marketplace** connecting customers with verified
technicians (plumbing, electrical, AC, painting, cleaning, and more). It earns a **15–20% commission**
per completed job. Launch is **website + WhatsApp-first**, starting in **Maadi**, with a mobile app later.

It is intentionally **operations-first, not tech-first**: prove demand and run jobs manually before
automating. Everything must be **bilingual (English + Arabic)**.

## The plan in two phases

- **Phase 1 — MVP:** homepage, service pages, technician directory + profiles, a hybrid booking form
  (book by service *or* by preferred technician), admin booking dashboard, WhatsApp integration, reviews.
  FixIt manually assigns technicians and keeps final assignment authority.
- **Phase 2 — Marketplace expansion:** customer accounts, technician portal, automated matching,
  in-app messaging, online payments, commission engine, loyalty/referrals, analytics.

See `website/deliverables-2-phases.md` for the exact deliverable list.
