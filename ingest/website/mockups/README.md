# Website mockups

Visual design references for the FixIt Egypt website. These are concepts/comps, not final assets.

- `homepage-mockup-en.jpeg` — **English** mockup. Three frames: (1) Homepage — hero "Reliable Home Services in Maadi", services grid, How It Works, Why Choose FixIt, testimonials, "Book on WhatsApp" CTAs, a "50 EGP off your first service" badge; (2) "Our Verified Technicians" listing with ratings, experience, jobs-completed, "Book This Technician"; (3) a WhatsApp automation flow (auto-reply → service selection → booking confirmation → technician assigned → after-service follow-up → review request).
- `phases-mockup-ar.png` — **Arabic** mockup showing both phases: a "PHASE 1 — MVP" row (request-a-service + manual management) and a "PHASE 2 — MARKETPLACE" row (fuller marketplace UI with technician cards, profiles, dashboard).

## How they map to the brief

- Confirm the **navy + orange** brand palette and the house-and-wrench logo lockup.
- Demonstrate the **hybrid booking** (book by service OR by technician) from `../deliverables-2-phases.md`.
- Show the **WhatsApp-first** operations model and the verified-technician trust framing.
- Establish that the site must ship **English + Arabic**.

> These are the design starting point. When real designs (Figma, etc.) exist, link the source here.
