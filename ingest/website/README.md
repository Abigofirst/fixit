# website

Website deliverables for Fixit.

**What lives here:**
- Page designs, mockups, wireframes
- Site copy (per page)
- Page specs / structure / sitemap
- Web-ready assets (images, icons exported for the site)
- Component or section notes

**Conventions:**
- Organize by page or section where it helps (e.g. `home/`, `about/`).
- Keep copy as markdown so it's easy to diff and edit.
- Note the intended source-of-truth for design (Figma link, etc.) if the design lives elsewhere.
