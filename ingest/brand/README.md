# brand

Brand identity material for Fixit.

**What lives here:**
- Brand guidelines (the master doc, plus any source PDFs/decks)
- Logos (all formats: SVG, PNG, vector source)
- Color palette (hex/RGB/CMYK values)
- Typography (font names, files, usage rules)
- Voice & tone, messaging, taglines
- Iconography, imagery style

**Conventions:**
- Keep a `brand-guidelines.md` as the human-readable single source of truth; link source files from it.
- Store logo variants in a `logos/` subfolder if there are several.
- Record exact color values as text (not just images) so they're searchable and copy-pasteable.
